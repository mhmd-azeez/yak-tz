## Yak Timezone Bot
Can convert date/time to any timezone you want. Very helpful for remote-first teams. Example queries:

```
/tz 18:40 CDT to UTC+3
```

```
/tz 2024-08-01 2:00 PM GMT => CEST
```

```
/tz convert 02:30 CET to GMT please
```
